vue router-link 点击时背景颜色问题

March 16, 2021 · 1 min read

zjiaahui

自学web前端的小白
note

Vue通过vue-clipboard2实现一键复制内容到剪贴板功能 在点击瞬间，会有淡蓝色的背景颜色 这不是 router-link-active 设定问题，通过设置 router-link-active 无效 ，淡蓝色依然存在在点击的瞬间

router-link默认会转换成 a 标签因此我们重置一下这个标签的样式就行
a,
a:hover,
a:active,
a:visited,
a:link,
a:focus {
-webkit-tap-highlight-color: rgba(0, 0, 0, 0);
-webkit-tap-highlight-color: transparent;
outline: none;
background: none;
text-decoration: none;
}
