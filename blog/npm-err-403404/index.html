npm发布包运行npm publish 报403 404问题

March 5, 2021 · 1 min read

zjiaahui

自学web前端的小白
摘要

我想上传我写的轮子到npm上，最后运行npm publish发布代码时，报错E403或者E404

npm publish 报404#

原因#

npm账号未注册

解决办法#

先去官网注册npm账号，然后本地项目下运行npm adduser进行注册（此时要注意，如果之前更换了npm源则需要改回官方npm源），如果npm publish还是出现404即再尝试一次npm adduser

npm publish 报403#

原因#

官网注册后未进行邮箱验证

解决办法#

我们在官网注册时，npm会发送邮件到你填写的邮箱，打开邮件按操作验证即可
