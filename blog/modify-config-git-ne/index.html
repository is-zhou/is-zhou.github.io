Git修改和配置本地用户名和邮箱

June 1, 2020 · 1 min read

zjiaahui

自学web前端的小白

本文将介绍如何修改当前设备上已经为git配置过的用户名和联系邮箱信息 ...

1、查看git所有配置#

  • 输入如下命令查看所有配置项
git config --list

2、查看git用户名邮箱#

  • 分别输入如下命令查看现有配置的用户名和邮箱
git config user.name
git config user.email

3、配置git用户名邮箱#

  • 如果还未配置用户名和邮箱可通过如下命令配置用户名和邮箱
git config user.name "你的用户名"
git config user.email "你的联系邮箱"

4、修改git用户名邮箱#

  • 如果已经配置过用户名和邮箱可通过如下命令分别修改
git config --global --replace-all user.name "你的用户名"
git config --global --replace-all user.email "你的联系邮箱"
